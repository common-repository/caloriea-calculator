== Description ==


**For using This plugin you Can Know That How Many calories You burn daily.**

**you can know the perfect amount of calories to eat if you want to maintain and lose your body weigth.**

To run this plugin you need to put shortcode in your page.
([calorie_calculator])


<h3>Features:</h3>

<ul>
<li>Easy to use</li>
<li>Select activity</li>
<li>Customize header title text and font size with title color</li>
<li>Set form title color and background color</li>
<li>Customize Field active and activity field hover color</li>
<li>Customize Calculate button text and background color</li>
<li>Customize button text hover color, background hover color</li>
<li>Set calorie message background and text color</li>
<li>Set title tab background and active background color</li>
<li>Change list background and text color</li>
</ul>
